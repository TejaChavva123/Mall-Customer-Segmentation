# Mall-Customer-Segmentation

# Problem Statement:

  **Here we are segmentation of customers coming to a mall based on thier annual income and thier spending score.**
  
# Algorithm used:

  **K Means Clustering**
  
# Method used to predict k:

  **Elbow Method**

# Python Libraries Used:

  **1) Numpy**
  
  **2) Pandas**
  
  **3) Matplotlib**
  
  **4) Seaborn**
  
  **5) Scikit-Learn**
